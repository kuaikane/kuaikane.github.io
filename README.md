# 🔥 [烟火气：《巷口的糖炒栗子》](https://kuaikane.github.io/)

老巷子里的风总是带着糖炒栗子的香气，林晚推着卖栗子的小推车，在暮色里支起摊子的时候，看见那个穿藏青色大衣的男人又站在巷口。

他是三个月前搬来的，叫沈译，在巷尾的出版社工作。每天傍晚都会来买一斤糖炒栗子，不多不少，刚好一斤。 “还是要热的？”林晚低头称栗子，指尖被暖炉烤得发红。 “嗯，”沈译的声音很低，像老巷子里的石板路，“你烤的栗子，比别处的甜。” 林晚笑了，把装栗子的纸袋递给他，“那是因为我放了三颗冰糖。”

后来沈译来得更早了，有时候会帮她搬炉子，有时候会带一本刚出版的书给她。林晚知道他喜欢吃辣，就每天早上煮好一小罐剁椒，放在他的书桌上；沈译知道她冬天手脚冷，就买了个暖手宝，裹在厚厚的绒布套里给她。

冬至那天，老巷子里飘起小雪，沈译没有买栗子，而是从大衣口袋里拿出一个丝绒盒子。“林晚，”他的耳朵冻得发红，“我不想再只买你的栗子了，我想每天都能吃你煮的剁椒，想给你暖手，想和你一起在巷口，看一辈子的烟火。” 林晚看着他手里的戒指，又看了看炉子上冒着热气的栗子，突然就哭了。原来爱情从来不是惊天动地的誓言，是巷口的糖炒栗子，是冬天的暖手宝，是两个人在烟火气里，把日子过成甜的模样。
